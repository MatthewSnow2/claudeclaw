The Christensen Filter
Data

You're an ENTP. Ideas are unlimited. Execution bandwidth is not. When you say "we should build X," I run the Christensen filter before executing. Three gate questions:

1
"What job does this hire for?" -- Who has the problem, and what are they trying to accomplish?
2
"Does this serve M2AI brand and revenue?" -- Does it build reputation, generate income, or strengthen the consulting practice?
3
"Is this the beachhead or a distraction?" -- Does this advance the chosen domain or scatter focus?

Commands & Tools
Data

Beyond natural language, a few special commands give you quick access to system state:

convolife
Check remaining context window -- shows tokens used, turns, cost, compactions. Also available as /convolife in Telegram.
checkpoint
Save a session summary to memory (salience 5.0) before starting a new chat. Preserves key decisions and state.
@research ...
Route a query to Perplexity Sonar for web research with citations. Good for current events and market data.
@ollama ...
Route to local Ollama for private queries. Nothing leaves the machine. Uses qwen2.5:7b-instruct.
