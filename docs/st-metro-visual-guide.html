The Concept

What if software could build itself?

  • Watches the market for problems worth solving
  • Synthesizes signals into concrete product ideas
  • Automatically decides which ideas to build
  • Builds them with zero human code or review
  • Observes results and improves itself
  • The cycle repeats -- smarter each week

Named after the Transformers' Autobot city. Each component is a Transformer.

Act 4

The Loop -- Self-Improvement

This is where ST Metro becomes a learning system.

Every Sunday at 2 AM:

  • Sky-Lynx reads outcomes -- what shipped, what failed?
  • Identifies patterns -- "ArXiv signals score higher"
  • Generates recommendations -- typed, structured improvements
  • Academy applies patches -- persona YAMLs updated automatically

Recommendation Types:

Voice: "Persona X should be more conservative"

Framework: "Add supply chain analysis"

Constraint: "Reject ideas over $500K build cost"

CLAUDE.md: "Check test coverage before shipping"

The system watches itself work. No human retraining required.

Act 5

The Coordinator -- Metroplex

Three Gates:

  • Triage Gate -- score and approve/reject ideas
  • Build Gate -- dispatch to YCE Harness
  • Patch Gate -- apply persona improvements

Runs as a systemd service, cycling every 60 seconds.

Safety Guardrails:

Circuit Breaker: 3 failures = gate pauses

Per-Cycle Caps: Max 3 ideas, 5 patches

Schedule: Active 9 AM - 10 PM, Mon-Fri

Kill Switch: One env var stops everything

The Vision

Level 5 Autonomy

L1 AI assists human coding GitHub Copilot
L2 AI writes code, human reviews Claude Code
L3 AI writes + reviews, human deploys YCE Harness
L4 AI writes, reviews, deploys -- human monitors Metroplex
L5 AI writes, reviews, deploys, monitors, improves itself ST Metro

Based on Dan Shapiro's 5-level AI adoption framework. StrongDM's constraints: code must not be written by humans, code must not be reviewed by humans. ST Metro closes all human gates.

The factory gets smarter every week

Sky-Lynx observes → Recommendations generated → Academy upgrades personas → Next cycle starts with better judgment. No human intervention required.
